## csv2cliboard

### convertiert folgendes
*	utf8 -> latin1
*	trennzeichen , -> \t
*	23.45 -> 23,45
*	02.20.2017 -> 20.02.2017


### bedienungsanleitung:

1.  https://github.com/bw1/csv2cliboard/raw/master/csv2cliboard.zip
        herunterladen
2.  csv2clipboard.zip auspacken
3.  die exe auf den desktop legen
4.  die *.csv datei über das icon ziehen und fallen lassen
5.  die zwischenablage in einem leeren excel arbeitsblatt kopieren (^C-V)
